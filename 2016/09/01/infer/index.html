内存泄漏检查工具-infer

2016-09-01 visited：
目录
  1. 使用
回到顶部

infer是facebook开源的一款支持ios和android双平台的静态代码扫描的工具，Infer 可检查 Android 和 Java 代码中的 NullPointException 和 资源泄露。除了以上，Infer 还可发现 iOS 和 C 代码中的内存泄露。

使用

1 环境安装见官网

2 android 项目分析

infer -- ./gradlew build

在两次分析之间运行 gradlew clean
